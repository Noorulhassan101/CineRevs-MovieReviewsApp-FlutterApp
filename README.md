# Zenthra

A premium, highly animated cinematic discovery platform built with Flutter.

## Overview
Zenthra is designed to provide an elegant, fluid, and modern user experience for discovering movies and anime. It abandons traditional, rigid app layouts in favor of dynamic gradients, smooth staggering animations, and a focus on high-quality aesthetics. 

## Features
- **Cinematic Interface**: Completely custom, premium UI utilizing `flutter_animate` for staggered entrance and interaction animations.
- **Media Discovery**: Browse, search, and discover movies and shows with rich details, cast information, and community reviews.
- **Community Engagement**: Create and join groups tailored to specific genres or shared interests.
- **Seamless Authentication**: Secure login and registration powered by Firebase, including one-tap Google Sign-in integration.
- **Cross-Device Sync**: All user preferences, favorites, and profile data are stored securely on Firestore, ensuring a consistent experience across sessions.

## Getting Started for Developers
To run this project locally:

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Ensure you have your Firebase `google-services.json` configured in the `android/app/` directory.
4. Run `flutter run` on your target device or emulator.
